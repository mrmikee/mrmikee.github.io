Remote Base

2020-01-01
Example Ham Shack (not my dad’s)

For a while now my father has been after me to help him build a system that will allow him to remotely operate his ham radio shack. I have used this project as my motivation to expand my comfort zone and to learn more about some of the technologies I have taken for granted for years.

  • VPN
  • TCP/IP networking
  • serial communications
  • audio interfaces
  • and more…

As I post articles on my site I will associate them to this project as appropriate for others to use as a reference and to help keep the bigger picture in focus.

More to come…you may have to wait a while things are really busy around here.
